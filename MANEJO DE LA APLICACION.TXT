Y VISUALIZA DISTRIBUCIONES DE LAS VARIABLES SELECCIONADAS.
PRIMERO, RENOMBRA LAS COLUMNAS DEL DATASET PARA MAYOR CLARIDAD Y LIMPIA VALORES NUMÉRICOS EN LA COLUMNA VALOR_TOTAL.
LUEGO CONVIERTE LAS VARIABLES CATEGÓRICAS EN CÓDIGOS NUMÉRICOS Y VALIDA SI LAS COLUMNAS REQUERIDAS ESTÁN PRESENTES. POSTERIORMENTE, TOMA UNA MUESTRA ALEATORIA DEL 10% DE LOS DATOS PARA CALCULAR Y MOSTRAR UNA MATRIZ DE CORRELACIÓN ENTRE LAS VARIABLES SELECCIONADAS USANDO UN HEATMAP. FINALMENTE, GENERA GRÁFICOS DE DISTRIBUCIÓN PARA CADA VARIABLE, DESTACANDO LA MEDIA Y LA MEDIANA EN CADA GRÁFICO PARA FACILITAR LA INTERPRETACIÓN.


------------------CLASE DE MAIN.PY
ESTA CLASE CARGA UN DATASET DESDE UN ARCHIVO CSV, MUESTRA UNA MATRIZ DE CORRELACIÓN ENTRE LAS VARIABLES NUMÉRICAS, SEPARA LOS DATOS EN DOS CLASES BASADAS EN UNA COLUMNA OBJETIVO Y GRAFICA LAS DISTRIBUCIONES DE LAS VARIABLES NUMÉRICAS. PRIMERE INTENTA LEER LOS DATOS Y LOS MUESTRA EN PANTALLA SI EL ARCHIVO EXISTE. DESPUÉS, CALCULA Y VISUALIZA UNA MATRIZ DE CORRELACIÓN UTILIZANDO UN HEATMAP. A CONTINUACIÓN, DIVIDE EL DATASET EN DOS SUBCONJUNTOS (CLASE 0 Y CLASE 1) BASADOS EN LA COLUMNA OBJETIVO ESPECIFICADA MOSTRANDO LA CANTIDAD DE FILAS EN CADA CLASE. FINALMENTE, GRAFICA HISTOGRAMAS PARA LAS VARIABLES NUMÉRICAS, PERMITIENDO ANALIZAR SUS DISTRIBUCIONES.


----------------CLASE makeDataSet
ESTE CÓDIGO GENERA UN DATASET SINTÉTICO CON VALORES NUMÉRICOS ALEATORIOS Y UNA COLUMNA  COMO LO REALIZO
EL INGENIERO EN LA CLASE DE HOY CON UNA DE CLASIFICACIÓN BINARIA (0 Y 1). PRIMERO, EL USUARIO DEFINE EL NÚMERO DE FILAS, COLUMNAS NUMÉRICAS Y EL RANGO DE VALORES ALEATORIOS. EL DATASET SE CREA UTILIZANDO NUMPY PARA GENERAR LOS DATOS NUMÉRICOS Y SE ALMACENA EN UN DATAFRAME DE PANDAS, INCLUYENDO UNA COLUMNA ADICIONAL LLAMADA "CLASIFICACIÓN" CON VALORES BINARIOS. Y AL FINALEL USUARIO TIENE LA OPCIÓN DE GUARDAR EL DATASET COMO UN ARCHIVO CSV, INGRESANDO EL NOMBRE DEL ARCHIVO. EL CÓDIGO INCLUYE IMPRESIONES EN PANTALLA PARA MOSTRAR UNA VISTA PREVIA DEL DATASET Y CONFIRMAR OPERACIONES.
